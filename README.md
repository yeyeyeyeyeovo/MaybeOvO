# 我的个人博客

这是我自己的个人博客网站！  
完全不需要编程知识，打开就能用。

---

## 怎么打开这个博客？

**最简单的方法**（推荐）：
1. 进入这个文件夹（`个人博客`）
2. 双击 `index.html` 文件
3. 它会自动在浏览器里打开

就是这么简单！

---

## 我想改成自己的内容，怎么办？

超级简单！只需要改一个文件里的几个地方。

### 第一步：打开文件
用**记事本**或**VS Code**打开 `index.html`

### 第二步：找到配置区域（最上面）
文件最上面有一大段用 `//` 开头的文字，里面有个 `blogConfig` 部分：

```js
const blogConfig = {
    name: "林夏",                    // ← 把这里改成你的名字
    title: "写作者 · 独立开发者",     // ← 改成你的身份（比如“学生”“程序员”“自由职业者”）
    shortBio: "喜欢用文字记录生活...", // ← 改成你的一句话介绍
    avatar: "https://picsum.photos/...", // ← 改成你的照片地址
    ...
}
```

把这些改成你自己的就行。

### 第三步：修改文章
再往下一点，有一个叫 `posts` 的地方，里面有 5 篇文章示例。

你可以：
- 修改现有文章的标题、日期、内容
- 删除你不想要的文章
- 自己添加新文章（复制一个就行）

---

## 想放自己的照片怎么办？

最简单的方法：
1. 把你的照片复制到这个文件夹里（比如叫 `我的照片.jpg`）
2. 在 `index.html` 里找到 `avatar:` 那一行
3. 把内容改成：`"./我的照片.jpg"`

保存后刷新浏览器就能看到啦！
